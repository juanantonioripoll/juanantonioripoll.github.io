"C#" (pronounced cii sharp in English) is a multi-paradigm programming language developed and standardized by Microsoft as part of its .NET platform, which was later approved as a standard by ECMA (ECMA-334) and ISO (ISO /IEC 23270). C# is one of the programming languages designed for the common language infrastructure.
The name C# was inspired by the sign signo ‚ôØ, which is read as sharp in English for musical notation. I t is a play on words, since "C#" means, musically speaking, where the symbol s√≠mbolo # indicates that a note (in this case C, represented by C) must be a semitone higher. This is a metaphor for the superiority of C# over its predecessor C++ and in turn alludes to the same metaphor that was devised to give C++ its name. In addition, the # symbol can be imagined as the union of four + symbols, thus continuing the sense of progression of C languages. You can find more information on Wikipedia
